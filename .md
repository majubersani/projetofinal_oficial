# Raizes do Brasil

## Descrição do Projeto
A **Plataforma  Indígena** é uma aplicação web desenvolvida para valorizar e divulgar a cultura indígena. Ela permite que artesãos vendam seus produtos, como cocares, bolsas, acessórios e instrumentos, de forma simples e segura. A aplicação oferece uma interface amigável, tornando a navegação fácil para todos os usuários.

---

## Funcionalidades
- Cadastro e autenticação de usuários.
- Cadastro, atualização, listagem e exclusão de produtos.
- Busca e filtragem de produtos.
- Visualização detalhada de cada produto.
- Dashboard simples para artesãos gerenciarem seus produtos.
- Segurança de acesso (com JWT para rotas protegidas, opcional).

---

## Tecnologias Utilizadas
- **Backend:** Python, Flask, SQLAlchemy  
- **Banco de Dados:** SQLite (ou outro à escolha)  
- **Autenticação:** JWT (JSON Web Tokens)  
- **Frontend:** HTML, CSS, JavaScript (ou framework se houver)  
- **Dependências adicionais:** Flask-Pydantic-Spec, Flask-JWT-Extended
